# Chatbox

Simple socket program using Python socket interface.

# Requirement
Install python3.6 

for ubuntu 
```sh
$ sudo apt-get install python3.6
```
for fedora
```sh
$ sudo yum install python3.6
```
socket is inBuild package in Python

# How to use
first run server.py.

```sh
$ python3 server.py
$ python3 client.py
```
